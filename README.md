# Medical Residency App

Medical residency preparation platform.

## Features
- **Smart PDF Selection**: Selects relevant documents based on filters.
- **Quiz Engine**: Full exam simulation.
- **Spaced Repetition**: FSRS algorithm for efficient review.
- **Dashboard**: Study progress tracking.

## Deploy
Deployed on Vercel with DigitalOcean PostgreSQL.
